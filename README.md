# Travel Assistant

Travel planning app built with Ionic React + Vite.

## Features

- Auth (register/login)
- Trip planning by origin/destination
- Weather (current + date range)
- Transport options
- Flights and accommodation quick links
- Entertainment with categories
- Emergency numbers
- Safety tips
- Finance planner
- Pre-trip checklist
- Budget advisor

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start backend API:
   ```bash
   npm run server
   ```
3. In another terminal start frontend:
   ```bash
   npm run dev -- --host
   ```
4. Open app:
   - Local: `http://localhost:5173`
   - Phone in same Wi-Fi: `http://<your-pc-ip>:5173`

## Scripts

- `npm run dev` - frontend dev server
- `npm run build` - production build
- `npm run preview` - preview build
- `npm run server` - backend API
- `npm run test.unit` - unit tests
- `npm run test.e2e` - e2e tests
